# Signalbuch Flash Card App

A couples-friendly flashcard study app built with Next.js + TypeScript + Tailwind CSS.

## Features
- Create and manage flashcard decks
- Add/remove cards with front & back sides
- Guided study sessions with card flipping
- Self-grading: Again / Hard / Good / Easy
- Session summary with score breakdown
- Light & dark theme support

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm start` | Start production server |
| `npm run lint` | Run ESLint |
